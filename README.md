# Great Houses Theme Collection

A collection of VS Code themes inspired by the great houses of ~~Hogw~~ code.

## Themes

- **House of Python**: Deep greens and shimmering silvers.
- **House of Gopher**: Rich blues and elegant bronze.
- **House of Camel**: Warm yellows and sturdy blacks.
- **House of Ruby**: Bold scarlets and brilliant golds.

Each theme is available in a standard and a "no-italics" version.

## Installation

1.  Open the **Extensions** sidebar in VS Code.
2.  Search for `Great Houses`.
3.  Click **Install**.
4.  Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and type `Preferences: Color Theme`.
5.  Select your preferred "House" theme.
